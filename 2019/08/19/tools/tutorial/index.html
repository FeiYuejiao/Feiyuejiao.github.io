    Building this website including 3 steps. First, use Hexo to setup local pages. Next, create github pages to launch your website. Then, write your own articles.

    Hexo

    Hexo is a fast, simple & powerful blog framework.

    Github Pages is a platform where you can host your website directly from your Github repository.

    Install Hexo

    $ npm install hexo-cli -g

    Init blog

    $ hexo init blog

    Now, there is a blog folder at where you init your blog.

    Choose the Theme

    You can choose whatever themes you like at https://hexo.io/themes/

    But here, we use ‘3-hexo’, which this blog uses.

    Download the theme

    $ cd blog/
$ git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo

    Configuration

    1. Modify the two places of _config.yml in the hexo root directory as follows
    theme: 3-hexo
highlight:
  enable: false # Turn off hexo rendering highlighting, instead with theme code block

    1. Add the avatar.jpg file as the avatar under the hexo root directory source

    2. Install the MUST INSTALL plugin

    $ npm i --save hexo-wordcount

    *Update

    Update the theme

    $ cd themes/3-hexo
$ git pull

    Hexo

    cd blog/

    $ hexo generate
$ hexo server

    Now Hexo is running at http://localhost:4000, Press Ctrl+C to open it.

    Github Pages

    First you should have your own github account.

    Configure Git Password Free: SHH and HTTPS

    SHH:

    $ cd ~/.ssh
$ ls
authorized_keys2  id_rsa       known_hosts
config            id_rsa.pub

    We need files called id_rsa or id_dsa, if you don’t have it, use ssh-keygen to generate it.

    $ ssh-keygen -t rsa -C "xx@xx.com"

    Copy ssh key in .pub file and paste it to Github->Setting->SSH and GPG keys. Press new SSH key.

    HTTPS Protocol

    Next, let’s talk about the more common HTTPS password-free method.

    $ touch ~/.git-credentials
$ vim ~/.git-credentials

    add content

    https://{username}:{passwd}@github.com

    add git configuration

    $ git config --global credential.helper store

    check ~/.gitconfig

    [credential]
    helper = store

    Now, git push should need no password any more.

    Create Github Repository

    Create a repository named your_github_id.github.io

    Install hexo-deployer-git Plugin

    $ npm install hexo-deployer-git --save

    Modify _comfig.yml, xxx is your username

    deploy:
   type: git
   repo: git@github.com:xxx/xxx.github.io.git
   branch: master

    Deploy

    $ hexo deploy

    Test

    Open your_github_id.github.io, should see your blog now!

    Shortcuts

    Add code below to ~/.bashrc 

    alias hs='hexo clean && hexo g && hexo s'  *# launch local server*
alias hd='hexo clean && hexo g && hexo d'  *# deploy your blog to github.io*

    Write Articles

    $ hexo new page "about"

    In ./source/_posts/ there should be about.md.
